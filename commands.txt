Docker compose:
-----
Dev
---
Bring up the devlopment env:
    docker compose -f docker-compose.dev.yml up -d --build

Bring up the production env:
    docker-compose -f docker-compose.prod.yml up -d --build

Delete volumes:
    docker compose down -v
-
DB
---
Connect to the db container terminal:
    docker compose exec db psql --username=db_admin --dbname=my_pg_db

Connect to db inside the db terminal:
    \c my_pg_db

create the table inside db, based on function at manage.py:
    docker-compose exec web python manage.py create_db
